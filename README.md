# Chaevie

Android-приложение «Калькулятор чаевых».

## Возможности

- ввод суммы счета с точностью до двух знаков после запятой;
- ввод процента чаевых от 5 до 30, по умолчанию используется 15%;
- ввод количества человек от 1 до 20, по умолчанию используется 1;
- выбор округления чаевых вниз, вверх или до ближайшего целого;
- расчет суммы чаевых, общей суммы и суммы на каждого человека с точностью до двух знаков;
- сообщения об ошибках для пустых, нечисловых, отрицательных и выходящих за пределы значений.

## Сборка

Проект подготовлен как Android-приложение на Java. Для сборки требуется установленный Android SDK.

```bash
gradle :app:assembleDebug
```
