# Chaevie

Android Studio-проект приложения «Калькулятор чаевых».

## Возможности

- ввод суммы счета с точностью до двух знаков после запятой;
- ввод процента чаевых от 5 до 30, по умолчанию используется 15%;
- ввод количества человек от 1 до 20, по умолчанию используется 1;
- выбор округления чаевых вниз, вверх или до ближайшего целого;
- расчет суммы чаевых, общей суммы и суммы на каждого человека с точностью до двух знаков;
- сообщения об ошибках для пустых, нечисловых, отрицательных и выходящих за пределы значений.

## Работа в Android Studio

1. Откройте папку проекта в Android Studio через **File → Open**.
2. Дождитесь Gradle Sync.
3. Запустите приложение на эмуляторе или физическом устройстве кнопкой **Run**.

Проект использует стандартную структуру Android Studio: модуль `app`, `AndroidManifest.xml`, XML-разметку экрана в `app/src/main/res/layout/activity_main.xml`, ресурсы в `app/src/main/res/values` и Java-код в `app/src/main/java`.

## Сборка из терминала

Для сборки требуется установленный Android SDK и доступ к Android Gradle Plugin.

```bash
gradle :app:assembleDebug
```
